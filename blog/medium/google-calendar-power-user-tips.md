# Google Calendar Tips Most Power Users Never Discover

*Originally published on [Schedule Calendar Blog](https://calendar-extension.site/blog/google-calendar-power-user-tips.html)*

---

Google Calendar has a substantial set of features below the surface level. Most users never discover them because the interface does not prompt exploration. These are the settings and habits that genuinely change how the tool feels to use.

Three settings to change immediately: Speedy meetings (Settings > General preferences — shortens default event durations to create automatic buffers), Working hours (signals availability and prevents out-of-hours invites), and Show declined events (uncheck this to declutter the view).

Keyboard shortcuts must be explicitly enabled: go to Settings > Keyboard shortcuts and turn them on. C creates a new event. T jumps to today. 1-5 switch between day, week, month, year, and schedule views. J and K move forward and backward in time. These significantly speed up full-tab calendar sessions.

Two event features most users miss: the Suggested times panel (appears when adding guests — surfaces mutual free windows without individually checking each person's calendar), and the Out of office event type (auto-declines invites and sends auto-replies — different from a regular event named 'OOO').

Multiple calendars by type — work, personal, travel — enable cleaner sharing and visibility control. Share the work calendar with colleagues without exposing personal events.

Set a conferencing default once in Settings > Add conferencing. New events automatically include the meeting link.
