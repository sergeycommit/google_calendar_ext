# Copy My Weekly Calendar Template (It Takes 20 Minutes to Set Up)

*Schedule Calendar Weekly · May 15, 2026*

---

One of the best things about time blocking is that you don't have to design the system from scratch every Monday morning. Once you have a template that works, you apply it, adjust for the specific week, and move on.

Here's the structure I use. Feel free to copy it directly.

## The core architecture

My week is divided into three types of time, and each gets a dedicated color in Google Calendar:

**Green: Deep work blocks.** 90–120 minutes, mornings only (8:30–10:00 or 9:00–11:00). Specific task name, Busy status, no notifications during. Maximum two per day.

**Blue: Meetings.** All external meetings, syncs, and calls. I try to cluster these into Tuesday afternoon and Thursday — keeping Monday, Wednesday, and Friday mornings cleaner.

**Orange: Shallow work batches.** 30–45 minutes, once in mid-morning and once after 4pm. Email, Slack, admin, expense reports, quick reviews.

## The weekly template (Monday–Friday)

**Monday:**  
8:30–10:00 — Deep work (most important project task)  
10:00–10:15 — Buffer  
10:15–11:00 — Shallow work batch  
Rest of day: open for meetings, follow-ups, planning

**Tuesday:**  
8:30–10:00 — Deep work  
10:00–12:00 — Meeting window  
12:00–12:30 — Buffer/lunch  
1:00–4:00 — Meeting window  
4:00–4:45 — Shallow work batch

**Wednesday:**  
8:30–10:00 — Deep work  
10:00–12:00 — Second deep work or focus project  
Afternoons: open

**Thursday:**  
Similar to Tuesday: meeting-heavy  
Morning deep work block still protected

**Friday:**  
8:30–10:00 — Deep work or weekly review  
Rest of day: intentionally lighter; used for planning next week

## What "template" actually means in practice

I set these blocks as recurring events in Google Calendar. They populate automatically each week. Then on Sunday evening or Friday afternoon, I do a 10-minute review: update block names for the specific tasks of the coming week, check for conflicts, move anything that needs to move.

The template doesn't run the week. It gives the week a structure to start from.

> Start with a simpler version than you think you need. Two or three protected deep work blocks per week is enough to see a difference. You can always add more after the habit is established.

**This week's takeaway:**
- Three categories (deep work, meetings, shallow work) are enough to start
- Recurring events create the template automatically — you just update the names weekly
- Start simple; two blocks per week is better than a complex system you'll abandon
