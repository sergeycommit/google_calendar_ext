# The Google Calendar Power User Tricks I Use Every Week

*Schedule Calendar Weekly · February 18, 2028*

---

This is a practical one. After two years of weekly tips about calendar habits and workflows, here's the collection of small Google Calendar features and tricks that have stuck — the ones I use consistently and that I recommend to anyone who works heavily in Google Calendar.

## The features power users know

**Drag to resize events.** You can extend or shorten any event by dragging its bottom edge. Faster than editing the event and changing the end time.

**Duplicate event.** Right-click any event → Duplicate. Creates an instant copy at the same time that you can then move and rename. Useful for creating a similar recurring block without starting from scratch.

**Copy event to another calendar.** Right-click → Copy to calendar. When you want to move an event to a different calendar layer (e.g., from work to personal) without losing any details.

**Batch color change.** Open an event, click the colored dot next to the event title to change color. Does not affect other events in the series unless you choose to.

**Create events by dragging in the calendar.** Click and drag in the calendar grid to create an event for that exact time block. Faster than the + button when you already know the time slot.

**Command/Ctrl + Z to undo.** Google Calendar supports undo for most actions. If you accidentally delete or move an event, undo gets it back.

**The ? key.** Opens the full keyboard shortcut reference. Most users have never pressed it.

## The workflow tricks

**Event description as a pre-meeting notes document.** Before a meeting, I paste my talking points into the event description. They're accessible from any device without needing a separate file.

**Attachment from Drive.** You can attach Google Drive documents directly to calendar events. Click the Google Drive icon in the event creation form. The document link appears in the invite for all attendees.

**Working location.** Set your working location (office, home, elsewhere) for each day via the "Working location" row that appears at the top of the day view when enabled.
