# Google Calendar Keyboard Shortcuts: Building Muscle Memory for Daily Operations

*Hacker News post · February 21, 2027*

---

Google Calendar has a full keyboard shortcut system that most daily users have never discovered. The ? key opens the shortcut reference — it's been there for years. The shortcuts are worth knowing because calendar navigation is a high-frequency operation where 20-second savings at 20× daily frequency compounds meaningfully.

## The shortcuts with the highest daily ROI

**Navigation:**
- `T` — Go to today. Most useful shortcut after drifting to a future date.
- `1` — Day view
- `2` — Week view  
- `3` — Month view
- `4` — 4-day view (useful for engineering sprint planning)
- `←/→` — Previous/next period in current view

**Event operations:**
- `C` — Create new event (full form)
- `Q` — Quick add (natural language parser: "meeting with Alex Friday 3pm")
- `E` — Edit selected event
- `Delete` / `Backspace` — Delete selected event
- `Esc` — Close popup or cancel

**Search:**
- `/` — Focus search bar

## The learning approach that actually works

Learning all shortcuts at once doesn't build muscle memory. The mechanics of habit formation favor narrow, repeated practice.

Pick 3 shortcuts that correspond to your 3 most frequent operations. Practice them exclusively for one week. Then add 2 more. The shortcuts you use become automatic within 5-7 days of deliberate practice; the ones you don't use remain unavailable regardless of how many times you read the list.

## The compound effect calculation

`T` (go to today): saves ~5 seconds vs. clicking through the calendar header. Used ~10x/day: 50 seconds/day, ~4 minutes/week. Not transformative, but multiplied across every shortcut you automate, it compounds.

The more important effect: shortcuts change the subjective experience of using the tool. Calendar navigation that's fluid and immediate feels qualitatively different from navigation that requires visual search and clicking.

## The shortcut that most people miss

`S` — Schedule view (list view). Shows upcoming events chronologically, without the visual grid. For quick "what's on the next few days" scanning, it's faster than any other view.
