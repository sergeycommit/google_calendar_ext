# Google Calendar Power User Features: The Non-Obvious Ones Worth Knowing

*Hacker News post · February 6, 2028*

---

After several years of heavy Google Calendar use, these are the features that aren't in the standard "tips" listicles but have compounding daily value.

## The non-obvious features

**Drag the bottom edge of an event to resize.**  
You don't need to open the event and change the end time. Click and drag the bottom edge of any event block in the weekly view. Saves 5-10 seconds per duration change; used frequently for planning adjustments.

**Ctrl/Cmd+Z undo.**  
Most people don't know this exists. Accidentally moved an event? Ctrl+Z. Accidentally deleted something? Ctrl+Z. Undos last several actions. Not in the Google Calendar documentation prominently.

**Copy event to another calendar via right-click.**  
Right-click any event → "Copy to" → choose calendar. Useful when you have an event on your personal calendar that belongs on your work calendar, or vice versa.

**Duplicate event via right-click.**  
Right-click → Duplicate. Creates a copy at the same time in the same view. Edit the copy immediately. Useful when creating a similar event (same attendees, same duration, different agenda).

**Event description as meeting notes.**  
The event description is visible to all attendees and syncs across devices. I use it for pre-meeting notes and talking points. Search-indexed, accessible from mobile. Replaces hunting for a separate notes doc.

**Google Drive attachment to events.**  
When creating an event: click the Google Drive icon in the "more details" view. Attach a document. It appears in the invite for all attendees. For spec reviews, design critiques, or pre-read documents — this makes the attachment visible in the calendar rather than requiring a separate email.

**Working location per-day setting.**  
Set your working location for each day: Office, Home, Somewhere else. Appears on your calendar visible to colleagues. Replaces "are you in the office today?" Slack messages.

**Time zone display options.**  
Settings → General → Time zone. You can display a secondary timezone column in the weekly view. If you have a regular meeting partner in a specific timezone, this eliminates the arithmetic on every scheduling interaction.

## The Workspace Admin features

For anyone who manages a Google Workspace: default sharing settings, working hours defaults, and domain-wide conferencing tool configuration propagate to all users. These deserve deliberate configuration rather than default acceptance.
