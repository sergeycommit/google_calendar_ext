# Google Calendar Settings Worth Changing (Most People Use Defaults)

*Hacker News post · March 21, 2027*

---

Google Calendar's default settings are mediocre compromises. They're designed for general cases, not for the specific ways that particular users actually work. 20 minutes spent in settings produces changes that persist silently for years.

## The settings worth changing

**Speedy meetings** (Settings → General → Event settings)
Default behavior: 60-minute meetings end at 60 minutes. Speedy meeting behavior: 60-minute meetings end at 50 minutes, 30-minute meetings end at 25 minutes. Creates automatic buffer without any per-meeting configuration. Recommended for anyone with more than 3 meetings/day.

**Default event duration** (Settings → General → Event settings)
Default: 60 minutes. Most knowledge worker meetings run shorter. Change to 30 minutes and adjust up when needed, rather than starting at 60 and running past it.

**Working hours** (Settings → Working hours)
Define start and end times for each day. This: (a) communicates availability to colleagues, (b) triggers a warning when someone tries to book you outside these hours, (c) is visible in shared calendar views. Essential for remote workers and anyone with non-standard hours.

**World clock** (Settings → General → World clock)
Add up to 4 time zones. They appear as a sidebar in the week view showing current local time. Essential for anyone scheduling across regions. Eliminates mental timezone arithmetic.

**Default notification timing** (Settings → Event settings → Notifications)
Default: 10 minutes before. Consider: 30 minutes before for any meeting requiring preparation, 5 minutes for routine meetings. Customizable per event category.

**Notification channels** (Settings → Event settings)
Default: desktop + mobile + email for the same event. Triple notifications for one reminder. Turn off email notifications if you check the calendar directly; they're redundant.

## The admin settings (Google Workspace)

If you're a Workspace admin: default sharing settings for the organization (free/busy vs. full details), default event visibility, and working hours defaults for new users. These propagate across the organization and are worth configuring during Workspace setup rather than leaving at Google defaults.
